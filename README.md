# OLIVE PEOPLE K-BEAUTY Website

Official first-version static website for **OLIVE PEOPLE K-BEAUTY**.

## Included

- Responsive mobile-first design
- English and Arabic language switch
- Full Arabic RTL layout
- About, Brands, Products, Membership, Store and Contact sections
- Search-engine metadata
- No framework or build process required

## Cloudflare Pages deployment

This website is intentionally made as a simple static site so it can be deployed from a phone.

### Build settings

- Framework preset: `None`
- Build command: leave empty
- Build output directory: `/`

### Files that must remain in the repository root

- `index.html`
- `styles.css`
- `script.js`
- `assets/logo.png`

## Details to replace before final public launch

1. WhatsApp number in `index.html`
2. Confirmed store address and opening hours
3. Final product names and prices
4. Confirmed supplier brands
5. Social media links, when available

## Brand direction

OLIVE PEOPLE focuses on authentic, useful Korean beauty products at fair prices. Avoid using “premium” as the main positioning language.
